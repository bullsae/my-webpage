# 생활 계산기 모음

연봉·퇴직금·실업급여·대출이자·평수 변환 등 실생활 계산기를 모은 무료 웹사이트입니다.
VSCode에서 수정 → GitHub push → Cloudflare Pages 자동 배포.

## 페이지
- `index.html` — 계산기 모음 메인(허브)
- `salary.html` / `salary.js` — 연봉 실수령액 계산기
- `toejik.html` — 퇴직금 계산기
- `sileop.html` — 실업급여 계산기
- `loan.html` — 대출이자 계산기
- `pyeong.html` — 평수 ↔ ㎡ 변환기
- `tools.css` — 공통 스타일

## 배포
```bash
git add .
git commit -m "수정 내용"
git push
```
